For Developers

Ship interactive 3D brain views in React—without building a 3D stack from scratch.

brain-for-web gives you a production-ready brain atlas component for web apps. Use npm install, TypeScript-friendly APIs, and Three.js-powered rendering to move from idea to demo fast.

Open Live Demo View GitHub Repo

Quickstart

npm install react-brain-atlas
import { BrainAtlas } from 'react-brain-atlas';

export default function App() {
  return (
    <BrainAtlas
      onRegionClick={(region) => console.log(region)}
      className="h-[560px] w-full"
    />
  );
}
